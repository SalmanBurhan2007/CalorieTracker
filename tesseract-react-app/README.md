# Tesseract React App

This project is a React application that utilizes Tesseract.js for optical character recognition (OCR). Users can upload images of food items, and the application will scan the images to extract text.

## Features

- Upload images of food items.
- Optical character recognition using Tesseract.js.
- Display scanned text in the application.

## Getting Started

To get a copy of this project up and running on your local machine, follow these steps:

### Prerequisites

- Node.js (version 14 or higher)
- Yarn (optional, but recommended)

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/yourusername/tesseract-react-app.git
   ```

2. Navigate to the project directory:

   ```
   cd tesseract-react-app
   ```

3. Install the dependencies:

   ```
   yarn install
   ```

   or if you prefer npm:

   ```
   npm install
   ```

### Running the Application

To start the development server, run:

```
yarn start
```

or

```
npm start
```

This will open the application in your default web browser at `http://localhost:3000`.

### Usage

- Use the file input to upload an image of a food item.
- Click the "Scan Image" button to start the OCR process.
- The scanned text will be displayed below the image.

## Built With

- [React](https://reactjs.org/) - A JavaScript library for building user interfaces.
- [Tesseract.js](https://github.com/naptha/tesseract.js) - A pure JavaScript port of the popular Tesseract OCR engine.
